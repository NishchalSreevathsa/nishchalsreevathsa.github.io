Technical Profile

// engineering background & cross-domain expertise

My path into security engineering began with a foundation in software development understanding systems from the inside out before learning to break and defend them. That dual perspective became my core advantage: I don't just monitor threats, I understand the underlying code paths and protocol mechanics that make exploits possible.

Today I operate as a true Purple Team practitioner simultaneously thinking like an attacker while architecting defensive systems that hold. My work spans SIEM pipeline optimization, smart contract vulnerability analysis, cloud governance frameworks, and the emerging discipline of AI-augmented security operations.

Across Web3/Blockchain environments, I've worked deep in smart contract audit workflows, DeFi protocol risk assessment, and Solidity security pattern enforcement. In Healthcare and Financial domains, I've implemented HIPAA/PCI-aligned data governance, threat modeling frameworks, and continuous vulnerability management pipelines.

My current focus: building LLM-powered security agents using Model Context Protocol (MCP) to automate SOC analyst workflows reducing manual triage overhead while keeping sensitive data strictly local through privacy-first architecture.
